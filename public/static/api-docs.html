Nodes API

GET /api/nodes

Получить список всех узлов или фильтровать по типу

Query параметры:

  • type (optional): artist, collector, gallery, bank, expert

Пример:

GET /api/nodes?type=artist
GET /api/nodes/:id

Получить информацию о конкретном узле

POST /api/nodes

Создать новый узел

Request Body:

{
  "node_type": "artist",
  "name": "Иван Иванов",
  "jurisdiction": "RU",
  "metadata": {"style": "Импрессионизм"}
}
GET /api/graph-data

Получить данные графа для визуализации (узлы и рёбра)

Artworks API

GET /api/artworks

Получить список всех произведений

GET /api/artworks/:id

Получить детальную информацию о произведении

POST /api/artworks

Создать новое произведение

Request Body:

{
  "title": "Закат над Волгой",
  "artist_node_id": "artist-1",
  "year": 2024,
  "style": "Реализм",
  "medium": "Масло на холсте",
  "current_fpc": 5000000
}

Analytics API

POST /api/analytics-extended/price-corridor

Расчет ценового коридора для произведения

Request Body:

{
  "artwork_id": "artwork-1",
  "period_months": 6
}

Response:

{
  "artwork_id": "artwork-1",
  "current_price": 15000000,
  "corridor": {
    "gallery_median": 15000000,
    "sales_median": 12750000,
    "lower_bound": 15000000,
    "upper_bound": 15000000,
    "spread_percentage": 15
  },
  "liquidity": {
    "rating": "medium",
    "comparable_sales": 0,
    "comparable_listings": 0
  }
}
POST /api/analytics-extended/market-factors

Анализ рыночных факторов (F1, F2, F3)

Request Body:

{
  "artwork_id": "artwork-1"
}

Media Hub API

POST /api/media-hub/analyze

Анализ медиа-материала с sentiment analysis

Request Body:

{
  "title": "Рекордная продажа картины",
  "content": "Произведение установило новый рекорд...",
  "source": "Artnet News",
  "entity_type": "artwork",
  "entity_id": "artwork-1"
}
POST /api/media-hub/price-impact

Оценка влияния медиа на цену произведения

GET /api/media-hub/trending?days=7

Trending произведения по медиа-активности

Graph Segmentation API

POST /api/graph-segmentation/by-chronology

Сегментация по хронологическим периодам

POST /api/graph-segmentation/by-style

Сегментация по стилю

POST /api/graph-segmentation/by-geography

Сегментация по географии

POST /api/graph-segmentation/multi-dimensional

Многомерная кластеризация

Export API

GET /api/export/:type?format=csv|json

Универсальный экспорт данных

Параметры:

  • type: nodes, artworks, transactions, validations
  • format: json или csv

Примеры:

GET /api/export/nodes?format=csv GET /api/export/artworks?format=json
